# Hapi Generator

Opinionated generator using [easy-generator](https://github.com/mglagola/easy-generator).

## Installation

```bash
npm install -g hapi-site-gen
```

## Usage

```bash
Usage:
  $ hapi <api|site> <output-dir>

Examples:
  $ hapi api my-awesome-api
  $ hapi site my-awesome-site
```
