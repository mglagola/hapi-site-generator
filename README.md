# Hapi Generator

Opinionated generator using [easy-generator](https://github.com/mglagola/easy-generator).

## Installation

```bash
npm install -g hapi-site-gen
```

## Usage

```bash
Usage:
  $ markg <api|site> <output-dir>

Examples:
  $ markg api my-awesome-api
  $ markg site my-awesome-site
```
